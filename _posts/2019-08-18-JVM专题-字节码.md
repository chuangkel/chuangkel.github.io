# 字节码

泛型，编译器会自动生成桥接字节码

```java
public interface SupperClass<T> {
    void method(T t);
}
```

```java
public class SubClass implements SupperClass<String> {
    @Override
    public void method(String s) {
        System.out.println(s);
    }
}
```
