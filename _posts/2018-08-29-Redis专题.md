Redis五种数据结构

Redis中所有数据结构都是key-value的，各个数据结构是value不同，value可为五种数据结构：

1. String
2. List
3. Set
4. ZSet(有序的Set)
5. Hash(和map有什么区别,没有区别，这里是数组加链表的数据结构)
